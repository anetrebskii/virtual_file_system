1. Установка сервера
Скомпилированный сервер лежит в папке "Server". Для установки и удаления windows-сервиса необходимо использовать install.bat и uninstall.bat соответственно. Предварительно, данные скрипты надо настроить: для этого в каждом из них нужно указать корректный путь к утилите InstallUtil.exe (на данный момент там указан путь по-умолчанию).

2. Настройка сервера
Конфигурационные данные сервера находятся в файле VFS.Server.WinService.exe.config. В этом файле конфигурируются настройки WCF для определения места размещения обработчика клиентских запросов.

3. Запуск сервера (windows-службы)
В списке windows-служб запустить службу с названием "VirtualFileSystem".

4. Запуск клиента
Клиент не требует дополнительных настроек. Достаточно просто запустить исполняемый файл VFS.Client.exe. Первой командой, которую нужно ввести в появившейся консоли клиента, является команда для подключения к серверу:

connect адрес_сервера:порт имя_пользователя
